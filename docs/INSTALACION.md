# Guía de Instalación - 8cars

Esta guía te llevará paso a paso desde cero hasta tener el proyecto corriendo localmente.

## Prerrequisitos

Antes de empezar, asegúrate de tener instalado:

1. **Node.js** (versión 18 o superior)
   - Descarga desde: https://nodejs.org/
   - Verifica instalación: `node --version`

2. **Git**
   - Descarga desde: https://git-scm.com/
   - Verifica instalación: `git --version`

3. **Un editor de código** (recomendado: VS Code)
   - Descarga desde: https://code.visualstudio.com/

## Paso 1: Configurar el Repositorio en GitHub

### 1.1 Crear repositorio en GitHub

1. Ve a https://github.com
2. Inicia sesión o crea una cuenta
3. Click en el botón verde "New" (Nuevo)
4. Configura el repositorio:
   - Nombre: `8cars`
   - Descripción: "Página web para 8cars - Automotora Digital"
   - Privado o Público (a tu elección)
   - NO marcar "Initialize with README" (ya tenemos uno)
5. Click en "Create repository"
